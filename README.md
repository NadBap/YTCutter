# YTCutter
---
**YTCutter** is a youtube downloader made by using python, command terminal, and yt-dlp. I plan on adding a way to clip downloaded video, but the downloader part is done for now



## HOW TO IMPORT OWN COOKIES
Please export your own cookies as this is a throwaway account and might not work if
used to much

**Step 1**: Download a cookies.txt extension on preferred browser

Firefox: https://addons.mozilla.org/en-US/firefox/addon/cookies-txt/ 

Google: https://chromewebstore.google.com/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc


**Step 2**: Sign into account and make sure you can play mix/age-restricted videos

**Step 3**: Press the extension and export cookies 

Firefox: Press on current site

Google: Press Export



**Step 4**: Rename the file cookies.txt

**Step 5**: Drag and drop cookies.txt into the Util folder and replace the prior cookies files

**Step 6**: Pray
